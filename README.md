# Проектное задание Neoflex  
Analytics DWH: VTB / BDS / NR / ФРиР


## Общее
На данном проектном задании учащийся может попробовать себя в роли аналитика и выполнить типовую проектную задачу.

Может попробовать себя в различных видах проектной активности: 

- Входит на проект и познает новую для себя среду; 
- Собирает и документирует требования гипотетического заказчика;
- Разрабатывает прототип (макет) на языке SQL (Structured query language); 
- Проектирует дашборд, показывающий результат в удобной визуальной форме;
- Презентует свою работу.
